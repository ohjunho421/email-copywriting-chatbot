# 포트원 플랫폼 정산 자동화 서비스 소개

## 서비스 개요
플랫폼/마켓플레이스 사업자를 위한 통합 정산 자동화 솔루션으로, 파트너 정산 자동 계산, 세금계산서 자동 발행, 안전한 지급대행을 하나의 플랫폼에서 제공합니다.

## 3대 핵심 서비스

### 1. 파트너 정산 자동화
**핵심 기능:**
- 주문건 기반 정산금 자동 계산 (파트너별 수수료 정책 자동 적용)
- 파트너 관리 (개인/법인/프리랜서 모두 지원)
- 정산 정책 등록 및 관리 (수수료율, 정산 주기 등)
- 사업자 휴폐업 조회 API
- 통장 자동 검증
- 사용자 정의 속성으로 원하는 데이터 필드 자유롭게 관리
- API 연동으로 정산 데이터 자동 생성
- ERP 시스템 연동 지원

**Pain Point 해결:**
- 한 달 걸리던 정산 업무를 이틀로 단축 (인프런 사례)
- 수기 정산금 계산 오류 제거
- 파트너별 다른 정산 주기와 수수료 체계 자동 관리
- 개인/법인/프리랜서 등 다양한 파트너 유형 통합 관리
- 휴먼 에러 제로

**실제 고객 사례:**
- 인프런: 한 달 걸리던 파트너 정산을 단 이틀 만에 완료
- IT 교육 플랫폼의 지식 공유자(파트너) 정산 자동화 성공

### 2. 세금계산서 자동화
**핵심 기능:**
- 정발행과 역발행 모두 지원 (홈택스는 역발행 절차가 복잡)
- 파트너 관리 페이지로 발행 대상 정보 저장 및 히스토리 관리
- 사업자 정보 조회 API로 자동 입력
- 휴폐업 조회 자동 확인
- 100,000건 이상 일괄 발행 가능 (홈택스는 최대 1,000건)
- 정산 데이터 기반 세금계산서 일괄 발행
- 이메일로 원클릭 승인 (공동인증서 로그인 불필요)
- 카카오톡 알림톡 연동 지원 (모바일에서 손쉽게 확인)
- 웹훅 지원으로 실시간 상태 확인

**홈택스 대비 차별점:**
- 홈택스 역발행: 송금내역서/사업자등록증 첨부 등 복잡한 절차 → 포트원: 간편한 역발행
- 홈택스: 발행 대상 저장 불가 → 포트원: 파트너 관리 페이지로 히스토리 관리
- 홈택스: 1,000건 제한 → 포트원: 100,000건 이상 대량 발행
- 홈택스: 매번 수기 입력 → 포트원: 정산 데이터 자동 연동

**Pain Point 해결:**
- 매달 수백~수천 건의 세금계산서 처리 자동화
- 수기 입력으로 인한 오류 방지
- 파트너와의 커뮤니케이션 비용 절감
- 세금계산서 발행, 취소, 오류 등 모든 상태 실시간 확인

### 3. 지급대행 서비스
**핵심 기능:**
- 365일 24시간 지급 가능 (주말/공휴일 포함, 은행 점검 시간 제외)
- 개인/법인/프리랜서 모두 지급 가능 (PG사는 법인만 가능)
- 지급 진행 중 취소 가능 (실시간 중간 취소)
- 예약 지급도 지급 직전까지 취소 가능
- 다중 PG사 정산금을 하나의 계좌로 통합 관리
- 과출금 방지 기능 (권장 출금 금액 자동 계산)
- 1,000건 이상 대량 일괄 지급
- 정산 내역서 자동 발송 (커스터마이징 가능)
- 회사 로고 및 도메인 설정

**PG사 지급대행 대비 차별점:**
- PG사: 법인사업자만 지급 → 포트원: 개인/프리랜서도 지급
- PG사: 영업시간에만 지급 → 포트원: 365일 24시간
- PG사: 바로지급 취소 불가 → 포트원: 실시간 취소 가능
- PG사: 예약지급 당일 전까지만 취소 → 포트원: 지급 직전까지 취소

**Pain Point 해결:**
- 제한적인 지급 시간 해결 (영업일뿐 아니라 주말/공휴일도 가능)
- 개인/프리랜서 지급 불가 문제 해결
- 오지급 사고 방지 (취소 기능)
- 다중 PG사 관리의 복잡성 해결

## 통합 자동화의 가치
**정산금 계산 → 세금계산서 발행 → 지급**을 하나의 플랫폼에서 완전 자동화합니다.

**주요 효과:**
- 한 달 걸리던 정산 업무를 이틀로 단축
- 월 수십 시간의 반복 작업 자동화
- 휴먼 에러 제거로 정확성 향상
- 재무팀 리소스를 핵심 업무에 집중
- 야근과 스트레스에서 해방

## 견적
**파트너 정산 자동화:** 별도 상담
**세금계산서 자동화:** 별도 상담
**지급대행:** 월 고정 비용 200,000원 + 송금 이체당 100원

## 국내 유일무이한 독보적 서비스
해외에는 Stripe Connect, Razorpay Route와 같은 플랫폼 정산 서비스가 있지만 국내에는 포트원이 유일합니다. 완벽한 정산 구현을 위해 수수료 및 정산 주기를 자동 계산하는 서비스는 더욱 찾아보기 어렵습니다.

## 적합한 대상
- 플랫폼/마켓플레이스 사업자
- 판매자/크리에이터/강사 등 파트너가 있는 비즈니스
- 매달 정산 업무로 어려움을 겪는 재무팀
- 빠르게 성장하면서 파트너 수가 증가하는 기업
- 정산 자동화로 운영 효율을 높이고 싶은 스타트업
